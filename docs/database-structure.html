Importante: Este proyecto NO utiliza Spatie Laravel Permission. Implementa un sistema personalizado y simplificado de roles.

Tabla principal de usuarios del sistema

  • id - INT UNSIGNED AUTO_INCREMENT
  • roles_id - INT (Relación 1:1 con roles)
  • ami_diamond, ami_gold, ami_silver - Flags de categorías
  • id_company - Relación jerárquica (empresa)
  • id_padre - Relación jerárquica (padre)
  • name - VARCHAR(191)
  • email - VARCHAR(191) UNIQUE
  • password - VARCHAR(191)

Características: Auto-referencias para jerarquía de usuarios, Soft Deletes, Flags personalizados para categorización

Resumen del Modelo de Negocio

GAF Solutions - Sistema de Gestión de Cartera

Modelo de negocio: Compra de carteras de créditos en mora y recuperación mediante descuentos directos por nómina (libranzas)

1. Compra de Carteras

Adquisición de portafolios de créditos en mora de entidades financieras

2. Gestión de Deudores

Principalmente pensionados del sector público con ingresos estables

3. Descuentos por Nómina

Recuperación mediante libranzas (descuentos directos)

4. Consultas a Centrales

Verificación de historial crediticio en CIFIN y DATACRÉDITO

5. Gestión de Embargos

Control de embargos judiciales que afectan la capacidad de pago
